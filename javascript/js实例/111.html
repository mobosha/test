return是向函数返回值，并终止函数的运行 比如： function a(){ return 10;//这个时候，函数a的值会等于10，并且函数中下面的内容不再执行 document.write(50);//由于上面已经return了，这条代码不会执行 } @@@@@@至于什么时候会用到，那看你要函数来做什么，你要他返回值就返回咯，不要他返回就不返回咯！@@@@ 比如，用函数来做一个加法运算，需要他返回值，那就这样 function a( b, c ) { return b + c; } 那么调用： var abc = a(5, 80); 这个时候，abc这个变量的值会是85 如果想直接输出结果，不要返回值那就 function a( b, c ) { document.write( b + c ); } 这个时候调用， var abc = a(5, 80); 那么页面会输出85，但abc是没有值的 当然，js中的return不一定非得用在函数function中，有时候也可以用来阻止某些动作， 比如表单的提交，让表单的提交事件返回false，那表单就不会提交：onsubmit="return false"
